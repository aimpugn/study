# useradd

## useradd?

`useradd` 명령어는 Linux 시스템에서 새로운 사용자를 추가하는 데 사용됩니다.
주어진 명령어 `useradd -u 1000 -g node -s /bin/bash -m node`는 특정 옵션과 함께 새로운 사용자를 생성하는 방법을 나타냅니다. 각 옵션의 의미를 자세히 설명하겠습니다:

## 명령어 및 옵션

```sh
useradd -u 1000 -g node -s /bin/bash node
```

`useradd -u 1000 -g node -s /bin/bash -m node` 명령어는 다음과 같은 설정을 사용하여 새로운 사용자를 생성합니다:

1. UID 1000을 가진 `node`라는 이름의 사용자 생성
2. 기본 그룹을 `node`로 설정
3. 기본 셸을 `/bin/bash`로 설정
4. 홈 디렉토리 `/home/node` 생성

- `-u 1000`:
    - 사용자 ID(User ID, UID)를 1000으로 설정합니다.
    - UID는 시스템에서 사용자를 식별하는 고유 번호입니다.
    - 일반적으로 시스템의 첫 번째 일반 사용자(루트 제외)의 UID는 1000입니다.

- `-g node`:
    - 기본 그룹을 `node`로 설정합니다.
    - `node` 그룹이 존재해야 하며, 이 그룹에 새로운 사용자가 속하게 됩니다.
    - 그룹 ID(Group ID, GID)는 이 그룹의 고유 번호입니다. 그룹은 파일과 디렉토리의 권한 관리에 사용됩니다.

- `-s /bin/bash`:
    - 사용자 셸을 `/bin/bash`로 설정합니다.
    - 사용자가 로그인할 때 기본적으로 사용할 셸을 지정합니다. 여기서는 `bash` 셸을 사용합니다.
    - `/bin/bash`는 Bourne Again Shell의 경로입니다.

- `-m`:
    - 사용자의 홈 디렉토리를 생성합니다.
    - 홈 디렉토리는 `/home/username` 형식으로 생성됩니다. 여기서는 `/home/node`가 됩니다.
    - `-m` 옵션이 없으면 홈 디렉토리가 생성되지 않습니다.

- `node`:
    - 생성할 사용자의 이름입니다.
    - 사용자 이름은 시스템에서 사용자를 식별하는 이름입니다.
