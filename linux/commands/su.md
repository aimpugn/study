# su

## su, switch user

## `sudo su -`

```sh
sudo su -
```

다음과 같은 방식으로 동작합니다.
- `su -`를 root 권한으로 실행합니다.
- root 사용자로 완전히 전환합니다.
- root의 홈 디렉토리로 이동합니다.
- root의 환경 변수를 로드합니다.

이는 다음과 같은 특징들을 갖습니다.
- `sudo -i`와 매우 유사한 동작을 합니다.
- 전통적인 방식으로, 일부 시스템에서 선호됩니다.
- root의 로그인 스크립트를 실행합니다.

### `-` (하이픈)의 의미

- 로그인 셸을 시뮬레이트합니다.
- 사용자의 환경을 완전히 초기화합니다.
- 대상 사용자(여기서는 root)의 환경 설정을 로드합니다.
