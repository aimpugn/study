## 사용 규약 (사람·AI 공용)

1. **책 전체 요약본은 만들지 않는다.** 알고리즘 지식 자체는 AI가 이미 갖고 있다. AI 세션에 필요한 것은 (1) 이 지도, (2) 학습자의 진행 상태 두 가지뿐이다.
2. 특정 주제를 다룰 때만 해당 챕터 페이지를 추출해, 책의 전개 방식(예시 전개, 점화식 유도 과정)을 참조한다.
3. 주제 노트는 **학습자가 그 주제 문제를 푼 뒤 자기 말로 쓸 때만** 이 폴더에 추가한다. 미리 만들어 두지 않는다.
4. 진행이 바뀌면 아래 표의 상태 칸을 갱신한다.

## 학습 루프 — 문제 먼저, 책은 조회

1. 주제 선정 → **책 펴기 전에** 그 주제 programmers 문제부터 시도 (타임박스 40분)
2. 막히거나 지저분하게 풀리면 → 그때 책의 해당 절만 읽는다. 반드시 질문을 들고 읽는다 ("내 풀이는 왜 비효율인가", "이 구조는 어떤 전이로 그 비용을 없애나")
3. 책 대표 알고리즘 1개를 책을 덮고 Java로 구현 (의사코드→Java 변환 자체가 전이 훈련)
4. 원래 문제 재풀이 + 같은 주제 1~2문제 추가
5. algorithm-retrospective 루틴으로 회고. "책의 어느 개념이 이 문제의 어느 결정에 쓰였나" 연결 한 줄 포함
