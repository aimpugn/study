단일 연결 리스트를 제자리에서 뒤집는 과정을 prev/cur/next 세 포인터와 노드 화살표의 단계별 변화로 보여주는 인터랙티브 시각화

풀이 시각화 — 화살표를 제자리에서 뒤집는 법

prev/cur/next 세 손이 리스트를 걸어가며 각 노드의 next 화살표를 뒤로 돌립니다. 프리셋을 고르거나 직접 입력해 단계를 넘겨 보세요. 파랑 = cur(지금 보는 노드), 초록 = 이미 뒤집힌 부분.

한 장면으로 기억하기: 다음을 먼저 붙잡는다(next) → cur의 화살표를 prev로 돌린다 → prev와 cur를 한 칸씩 민다.
핵심은 cur.next = prev로 돌리는 순간 원래 다음 노드로 가는 길이 끊긴다는 것. 그래서 next로 미리 붙잡아 둬야 나머지를 잃지 않습니다. cur가 null이 되면 prev가 새 머리입니다.
또는 직접:
prev (뒤집은 머리)
null
cur (지금 보는)
-
next (붙잡아 둔)
