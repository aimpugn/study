left/right 두 끝과 lastSeen 맵이 단계마다 어떻게 움직이는지. 프리셋을 고르거나 직접 입력해 단계를 넘겨 보세요. 파랑 = 현재 창, 빨강 = 곧 버려질 옛 글자.

한 장면으로 기억하기: 오른손(right)이 글자를 집는다 → 방명록(lastSeen)에서 "전에 본 적?"을 본다 → 그게 창 안(prev >= left)이면 왼손(left)을 그 다음으로 점프.
